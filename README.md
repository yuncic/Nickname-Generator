# Nickname Generator

### 우테코 닉네임 정하다가 정신이 나가버려 만들었습니다. 
* 바이브코딩으로 진행함

한글 자모 조합 기반 닉네임 생성기입니다.  
옵션(테토/에겐, 글자수, 금칙어 필터 등)에 따라 다양한 스타일의 닉네임을 생성하고, 결과를 선택/복사할 수 있습니다.

---

## 데모(배포 주소)
- Netlify: https://nickname-generator-two.vercel.app/

---

## 주요 기능

### 1) 닉네임 생성
- 한글 자모(초성/중성/종성) 조합으로 닉네임을 생성합니다.
- 옵션에 따라 “부드러운 느낌 / 또렷한 느낌 / 시원한 느낌” 등 음운 패턴을 반영합니다.

### 2) 테토/에겐 스타일 옵션
- 테토력 뿜뿜(남성스러운, 거친)
  - 초성: ㄱㄷㅂㅅㅈㅋㅌㅍ 비중 증가
  - 중성: ㅏ/ㅗ/ㅜ 위주
- 에겐력 뿜뿜(부드러움, 여성스러운)
  - 초성: ㅇㄴㄹㅁ 위주
  - 중성: ㅣ/ㅐ/ㅔ 비중 증가
- 튀는 느낌 제거
  - 중성에서 ㅘ/ㅚ 제외

### 3) 글자수 설정
- 생성할 닉네임의 길이를 설정할 수 있습니다.

### 4) 금칙어 필터
- 금칙어가 포함된 결과는 제외하거나 대체 생성합니다.

### 5) 다중 선택 & 복사
- 생성된 닉네임을 여러 개 선택해서 한 번에 복사할 수 있습니다.
